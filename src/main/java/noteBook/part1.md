### 프로젝트 파일들 설명
- src : 실제로 코드짜는곳
- resources 폴더 : html, css, javascript이미지 등 하찮은 파일 보관소
- application.properties : 환경변수나 프로젝트 설정 보관
- build.gradle : 프로젝트 이름, 버전, 라이브러리 등 기록용
- gradle 어쩌고 : gradle이라는 작업자동화 프로그램 돌리는거 도와주는 파일
